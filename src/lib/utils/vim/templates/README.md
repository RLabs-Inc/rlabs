# ${themeName} Vim Theme

This theme was generated using sacred geometry principles by RLabs Inc.

## Installation

1. Copy the theme file to your Vim colors directory:

   ```bash
   cp ${themeName}.vim ~/.vim/colors/
   ```

2. Add to your `.vimrc`:
   ```vim
   colorscheme ${themeName}
   ```

For Neovim:

1. Copy the theme file to your Neovim colors directory:

   ```bash
   cp ${themeName}.vim ~/.config/nvim/colors/
   ```

2. Add to your `init.vim`:
   ```vim
   colorscheme ${themeName}
   ```

- Full Treesitter support
- LSP integration
- Coc.nvim compatibility
